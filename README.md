# 🎓 Student Marks Predictor

## 📌 Project Overview
This is a beginner Machine Learning project that predicts student marks based on the number of study hours using Linear Regression.

## 🚀 Technologies Used
- Python
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## 📊 How It Works
The model is trained using sample study hours and corresponding marks.
It predicts marks for a given number of study hours and displays a regression graph.

## 🎯 Learning Outcome
- Understood basics of Linear Regression
- Learned how to train a ML model
- Uploaded and managed project using GitHub
